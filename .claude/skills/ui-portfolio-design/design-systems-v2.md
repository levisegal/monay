# UI Portfolio Design Systems (v2)

## 10 New Design Directions

### 1. üåÉ Neon Cyberpunk
**For**: Gaming platforms, crypto apps, futuristic products
**Style**: Dark backgrounds with vibrant neon accents, glowing effects

#### Typography
- Primary Font: Orbitron (futuristic), Rajdhani (sharp geometric)
- Scale: 14/16/20/24/32/48px
- Weights: 400, 600, 700
- Line Height: 1.4
- Letter Spacing: 0.05em (wide)

#### Color Palette
- Primary: #00ffff (Cyan), #ff00ff (Magenta)
- Secondary: #00ff00 (Neon Green), #ff0080 (Hot Pink)
- Background: #0a0a0f (Near Black), #1a1a2e (Dark Blue)
- Accents: #0096ff (Electric Blue), #ffcc00 (Neon Yellow)
- Text: #e0e0e0 (Light Gray), #ffffff (White)

#### Spacing System
- Base Unit: 8px
- Component Padding: 16-24px
- Card Spacing: 24-32px
- Generous margins for drama

#### Component Styles
- Borders: 1-2px with neon glow (box-shadow: 0 0 10px color)
- Border Radius: 0-4px (sharp, minimal)
- Shadows: Colored glows (box-shadow: 0 0 20px rgba(0, 255, 255, 0.5))
- Inputs: Dark with neon borders on focus
- Buttons: Solid backgrounds with glow effects on hover

#### Motion & Interaction
- Duration: 300-400ms
- Easing: ease-out
- Hover: Glow intensity increase, scale 1.02
- Focus: Pulsing neon outline
- Loading: Neon scanning animations

#### Anti-Patterns
- ‚ùå Light backgrounds (kills the neon effect)
- ‚ùå Muted colors
- ‚ùå Large rounded corners
- ‚ùå Subtle shadows

### 8. ‚ôø High Contrast Accessibility
**For**: Government sites, healthcare, education, universal access
**Style**: Maximum readability, clear focus states, WCAG AAA compliant

#### Typography
- Primary Font: Atkinson Hyperlegible (designed for readability), Open Sans
- Scale: 16/18/20/24/32/40px (minimum 16px body)
- Weights: 400, 700 (clear distinction)
- Line Height: 1.5 minimum
- Letter Spacing: 0.02em (improved readability)

#### Color Palette
- Primary: #0000ee (Pure Blue - standard link)
- Secondary: #000000 (Pure Black)
- Accent: #007700 (Dark Green - sufficient contrast)
- Backgrounds: #ffffff (Pure White), #f0f0f0 (Light Gray)
- Text: #000000 (Black on White)
- Focus: #ff6600 (Orange - 4.5:1 minimum)
- Error: #cc0000 (Dark Red)
- Success: #006600 (Dark Green)

#### Spacing System
- Base Unit: 8px
- Component Padding: 16-24px (sufficient touch targets)
- Minimum touch target: 44x44px

#### Component Styles
- Borders: 2-3px solid, high contrast
- Border Radius: 4-8px (subtle, clear corners)
- Shadows: Minimal, only for depth cues
- Focus: 3px solid outline, never removed
- Inputs: Clear borders, large labels
- Buttons: High contrast, clear active/focus states

#### Motion & Interaction
- Duration: Respects prefers-reduced-motion
- Easing: ease-in-out (predictable)
- Hover: Underline, background change (not just color)
- Focus: Always visible, high contrast outline
- Loading: Clear progress indication (not just spinners)

#### Anti-Patterns
- ‚ùå Low contrast text (below WCAG AA)
- ‚ùå Color as only indicator
- ‚ùå Small touch targets
- ‚ùå Removing focus outlines
